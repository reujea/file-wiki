### 문제 1 — 추정 키워드 stale (메타 룰 18 자기 적용 누락)

### 문제 2 — 메타 룰 1 sub-rule 카테고리 표 정보 부족

Phase 94에서 19건 누적 임계로 7 sub-rule(1a~1g) 분리 완료. 단 각 카테고리에:
- 해소 패턴 (어떻게 단일화하는가)
- 자동화 도구 (어떤 스크립트로 검출하는가)
- 신규 작업 진입 시 sub-rule별 분기 (어느 체크리스트 적용)

이 미정형화. 신규 작업자가 sub-rule 매칭 후에도 다음 행동 불명확.

### 문제 3 — 포트 매핑 단일 진실원 선언 누락

`spec/domain-map.md`가 사실상 포트 trait + 메서드의 단일 진실원으로 작동 중. 단 `spec/architecture.md:1716`의 LLMPort 한 줄이 "수치 표"인지 "메서드 매핑"인지 의도 불명. 신규 포트 추가 시 어디부터 갱신해야 할지 모호.

메타 룰 19 정식 승격(Phase 94) 이후 5건 누적했지만 spec 문서 내 포트 영역 자기 적용 미실시.
