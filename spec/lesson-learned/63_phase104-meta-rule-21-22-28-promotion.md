## 문제

### 문제 1 — 메타 룰 21 정식 승격 시 후보 섹션 중복 (메타 룰 19 자기 위반 회귀)

Phase 103에서 "## 메타 룰 21: ..." 정식 섹션 추가했으나 "## 메타 룰 21 후보: ..." 본문 섹션 삭제 누락. META.md `grep -cE "^## 메타 룰 [0-9]+:"` 결과 25 + `grep -cE "후보:"` 결과 5 — 메타 룰 21이 정식 + 후보 양쪽 등장. **메타 룰 19 (단일 진실원) 자기 위반 회귀 1건**. 메타 룰 25 (자기 적용 의무) §체크리스트 누락.

### 문제 2 — 메타 룰 22 누적 사례 본문 stale

본문에 누적 2건만 기재 (lesson 51 Phase 92 + lesson 53 Phase 94). 실제 누적:
- Phase 92 외부 협업 보류
- Phase 94 헥사고날 정공법 A vs B
- Phase 100 Settings IA 옵션 합의 (lesson 59)
- Phase 103 GraphRAG 흡수 옵션 합의 (lesson 62 본 phase 직전)

**4건 누적**이지만 본문 표 갱신 누락. 메타 룰 12 (잔존 N건 종결 의무) + 메타 룰 25 (자기 적용 의무) 양쪽 위반.

### 문제 3 — 메타 룰 28 정식 승격 가능 시점 미실행

Phase 101 등재 시 "Phase 102 정식 승격 가능" 명시했으나 Phase 102/103 동안 정식 승격 미수행. 메타 룰 25 자기 적용 의무 누락.
