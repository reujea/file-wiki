# personal-os-skills 분석

URL: https://github.com/ArtemXTech/personal-os-skills (408 stars)

## 개요

Obsidian + Claude Code CLI 스킬 모음 (Python 100%). 7개 스킬:
- **granola**: Granola → Obsidian 회의록 동기화
- **wispr-flow**: 음성 받아쓰기 통계/검색/내보내기
- **tasknotes**: Obsidian 태스크 API 연동
- **notebooklm**: 팟캐스트 → 프로토콜 변환 + 인용
- **notebooklm-import**: NotebookLM → Obsidian KG 임포트
- **recall**: 시간 기반 세션 검색 + 그래프 시각화
- **sync-claude-sessions**: Claude Code 대화 → Obsidian 마크다운 자동 동기화

## file-pipeline 관련성

**통합 불필요.** Obsidian 생태계 특화 도구로 아키텍처/코드 수준의 겹침 없음.

참고 가치:
- `recall` 스킬의 시간 기반 검색 + 그래프 시각화 UX → KG 검색 UI 개선 시 참고
- `sync-claude-sessions`의 세션 마크다운 변환 패턴 → 문서 전처리 아이디어
